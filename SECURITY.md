# Security Policy

## Supported scope

Security-sensitive issues should be reported for the currently deployed Excalvdraw application and the active default branch of this repository.

## How to report a vulnerability

Please do not post security-sensitive details in a public GitHub issue.

If GitHub private vulnerability reporting is available for this repository, use that first.

If it is not available, contact the maintainer through the contact information listed on the GitHub profile associated with this repository and include:

- a short summary of the issue
- affected area
- reproduction steps
- impact
- whether the issue is already public anywhere

## What to include

Helpful reports include:

- browser and version
- operating system
- whether the issue involves file handling, sharing, recording, or export
- proof of concept or redacted reproduction steps
- any suggested mitigation

## Disclosure

Please allow reasonable time to investigate and fix the issue before public disclosure.
